What is the current ratio?
the liquidity ratio is a measurement of a firm's ability to pay it's short term obligations with it's current assets.
How to calculate current ratio?
To calculate current ratio you should use the following formula:
current ratio = current assets / current liabilities

current ratio = current assets / current liabilities
